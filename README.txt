"cifraDeCesa_Criptografar" e "cifraDeCesa_Descriptografar" Como o proprio nome ja diz, esses programas criptografa ou descriptografa palavras ou frases.
    OBS: Não ultileze acentos nas palavras

Você irar precisar do node.js e a biblioteca readline-sync para rodar esses scrips


Para baixar o node, apenas acesse esse site e baixe
node.js: https://nodejs.org/en/

Para baixar o readline-sync abra seu terminal/cmd na pasta do programa e digite:
npm install readline-sync

Para executar o cifraDeCesa_Criptografar:
Abra o terminal/cmd na pasta do programa
Digite: node cifraDeCesa_Criptografar.js

Para executar o cifraDeCesa_Descriptografar:
Abra o terminal/cmd na pasta do programa
Digite: node cifraDeCesa_Descriptografar.js
